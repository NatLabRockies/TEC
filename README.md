
# Thermomechanical Simulation for CSP Receivers

This repository provides a Python script to simulate the thermomechanical behavior of Concentrating Solar Power (CSP) receivers under periodic thermal loading. The simulation couples elastic and creep strains to predict stress relaxation, fatigue, and rupture life of the receiver structure.

## Features

- **Thermomechanical Coupling:** Simulates both elastic and creep strain evolution in 3D receiver geometries.
- **Fatigue and Rupture Life Prediction:** Calculates fatigue and creep damage factors based on temperature and stress cycles.
- **Custom Mesh and Temperature Profiles:** Supports user-provided mesh and temperature data for flexible geometry and loading scenarios.
- **Postprocessing:** Outputs results as XDMF and PVD files for visualization, and saves key metrics as text files and plots.

## Requirements

- Python 3.x
- [FEniCS](https://fenicsproject.org/) (for finite element analysis)
- mshr (for mesh handling)
- numpy, scipy, matplotlib (for data processing and plotting)

Install dependencies (example for FEniCS on Docker):
```bash
docker run -ti -v $(pwd):/home/fenics/shared -w /home/fenics/shared quay.io/fenicsproject/stable:current
pip install numpy scipy matplotlib
```

## Input Files

- `mesh_1mm.xml`: 3D mesh of the receiver (generated by Gmsh or similar tools).
- `mesh_1mm_facet_region.xml`: Facet markers for boundary conditions.
- `real_temp_hours_*.txt`: Temperature profiles at different hours, with each file containing lines of `[x, y, z, T]` (coordinates in mm, temperature in K).

## Main Script

- `creep_fatigue_srlife_equations_backward_Euler.py`: Main simulation script. Reads mesh and temperature data, sets up material properties, applies boundary conditions, solves for displacement and stress, and computes damage factors over time.

## Output

- Results are saved in a directory named `results_<n_days>_days_with_srlife_VM_backward_740H/`:
  - XDMF/PVD files for mesh, temperature, displacement, stress, strain, and damage factors.
  - Text files with maximum values over time: `max_VM_stress_hdays.txt`, `max_VM_strain_hdays.txt`, `max_VM_CR_strain_hdays.txt`, `max_D_c.txt`, `max_D_f.txt`.
  - Plots of maximum equivalent stress, strain, and creep strain over time.

## Usage

1. **Prepare Mesh and Facet Files:**  
	Generate your mesh and facet marker files using Gmsh or another mesh generator.

2. **Prepare Temperature Profiles:**  
	Provide temperature profile files named `real_temp_hours_0.txt`, `real_temp_hours_1.txt`, ..., each containing tab-separated columns:  
	```
	x(mm)    y(mm)    z(mm)    T(K)
	```

3. **Run the Simulation:**  
	```bash
	python3 creep_fatigue_srlife_equations_backward_Euler.py
	```

4. **View Results:**  
	- Use ParaView or similar tools to visualize XDMF/PVD files.
	- Inspect text files and plots for maximum stress, strain, and damage metrics.
